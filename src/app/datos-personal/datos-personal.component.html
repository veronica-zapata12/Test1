Datos Personales

**Todos los campos son obligatorios**

el programa es requerido.
el nombre es requerido.
el nombre sólo debe tener letras, sin números, ni comas(,), ni puntos(.). 
el documento es requerido.
el documento debe tener minimo 5 números.  
el documento debe tener máximo 20 números.  
el documento no debería espacios, ni comas(,), ni puntos(.). 
el género es requerido.
